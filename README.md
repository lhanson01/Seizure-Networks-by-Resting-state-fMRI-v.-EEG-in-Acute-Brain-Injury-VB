This github repository contains the analysis code for the paper Seizure Networks by Resting-state fMRI v. EEG in Acute Brain Injury by Reuther et al. 

Analysis contains code for downloading data in ABI_Data_Download.R, cleaning data in ABI_clean_data.R, the functions used for analysis and the analysis itself in ABI_analysis.R, and the code for creating visualizations in ABI_visualization.R

Visualizations contain the created visualizations.
